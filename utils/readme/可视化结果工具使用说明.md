# 可视化工具使用说明

## 文件说明

`visualize_results.py` - 用于读取test.py生成的.pt文件并进行可视化和分析

## 功能

1. **可视化指定试验**: 绘制预测值vs真值的对比图
2. **保存指标摘要**: 生成包含所有试验详细信息的txt文件，支持排序

## 使用方法

### 1. 可视化单个试验

```bash
python visualize_results.py \
    --pt_path logs/trained_TCN_default/0/predictions_and_labels.pt \
    --trial_idx 5
```

**效果**：
- 生成 `visualization_trial_5.png`
- 显示该试验的所有关节的预测vs真值对比图

### 2. 保存指标摘要（不排序）

```bash
python visualize_results.py \
    --pt_path logs/trained_TCN_default/0/predictions_and_labels.pt \
    --save_summary
```

**效果**：
- 在 `logs/trained_TCN_default/0/metrics_summary/` 目录下
- 为每个label_name生成一个txt文件
- 文件包含所有试验的详细信息（原始顺序）

### 3. 保存指标摘要（按RMSE排序）

```bash
python visualize_results.py \
    --pt_path logs/trained_TCN_default/0/predictions_and_labels.pt \
    --save_summary \
    --sort_by rmse
```

**效果**：
- 按RMSE升序排列（最好的在前）
- 可以快速找到性能最好/最差的试验

### 4. 保存指标摘要（按R²排序）

```bash
python visualize_results.py \
    --pt_path logs/trained_TCN_default/0/predictions_and_labels.pt \
    --save_summary \
    --sort_by r2
```

**效果**：
- 按R²降序排列（最好的在前）

### 5. 同时可视化和保存摘要

```bash
python visualize_results.py \
    --pt_path logs/trained_TCN_default/0/predictions_and_labels.pt \
    --trial_idx 10 \
    --save_summary \
    --sort_by rmse \
    --output_dir ./analysis_results
```

**效果**：
- 可视化第10个试验
- 保存排序后的指标摘要
- 所有输出保存到 `./analysis_results/` 目录

## 参数说明

| 参数 | 必需 | 说明 | 示例 |
|------|------|------|------|
| `--pt_path` | ✅ | 测试结果.pt文件路径 | `logs/.../predictions_and_labels.pt` |
| `--trial_idx` | ❌ | 要可视化的试验序号 | `5` |
| `--save_summary` | ❌ | 是否保存指标摘要 | 添加此标志 |
| `--sort_by` | ❌ | 排序依据 | `rmse`, `r2`, `mae_percent` |
| `--output_dir` | ❌ | 输出目录（默认与pt文件同目录） | `./results` |

## 在Python脚本中使用

```python
from visualize_results import load_test_results, visualize_trial, save_metrics_summary

# 加载数据
estimates, labels, names, metrics = load_test_results('path/to/predictions_and_labels.pt')

# 可视化第10个试验
visualize_trial(estimates, labels, names, trial_idx=10, save_path='trial_10.png')

# 保存摘要
label_names = list(metrics['All'].keys())
save_metrics_summary(metrics, label_names, save_dir='./summary', sort_by='rmse')
```

## 注意事项

1. **序号对应关系**: 
   - `Original_Idx` 是原始的试验序号，用于可视化
   - `Idx` 是排序后的序号

2. **关节名称**: 
   - 代码中默认使用 `['Hip', 'Knee', 'Ankle']`
   - 根据实际情况修改 `visualize_trial()` 函数中的 `joint_names`

3. **采样率**: 
   - 代码假设200Hz采样率（5ms/样本）
   - 如需修改，调整 `visualize_trial()` 函数中的 `time` 计算

4. **文件大小**: 
   - txt摘要文件很小（每个通常<100KB）
   - 图片文件取决于序列长度（通常1-3MB）

## 常见问题

**Q: 如何批量可视化多个试验？**

A: 使用Python脚本循环调用：
```python
from visualize_results import load_test_results, visualize_trial

estimates, labels, names, _ = load_test_results('path/to/predictions_and_labels.pt')

# 可视化前10个试验
for i in range(10):
    visualize_trial(estimates, labels, names, i, f'trial_{i}.png')
```
